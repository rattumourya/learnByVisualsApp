# Learn By Visuals App

This repository hosts the **Animated CS Coach** front-end scaffold inside the `animated-cs-coach/` directory. It provides a Vite + React + TypeScript application with Tailwind CSS, shadcn/ui, Framer Motion, Zustand, and React Router already configured.

## Prerequisites
- Node.js 18+
- npm 9+

## Getting Started
1. Install dependencies:
   ```bash
   cd animated-cs-coach
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
   The app is served at [http://localhost:5173](http://localhost:5173).

## Available Scripts
From within `animated-cs-coach/` you can run:
- `npm run dev` – start the local dev server
- `npm run build` – create a production build in `dist/`
- `npm run preview` – preview the production build locally
- `npm run lint` – run ESLint checks
- `npm run test` – execute Vitest unit tests
