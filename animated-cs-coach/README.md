# Animated CS Coach

Animated CS Coach is a Vite + React + TypeScript starter kit for building interactive, animated computer science lessons across three tracks: Data Structures & Algorithms, System Design & Patterns, and OOP Concepts. It includes a reusable animation player, track-based routing, Tailwind + shadcn/ui styling, and scaffolding for adding new lessons quickly.

## Getting Started

```bash
npm install
npm run dev
```

The development server runs at http://localhost:5173.

### Build & Preview

```bash
npm run build
npm run preview
```

### Lint & Test

```bash
npm run lint
npm run test
```

## Project Highlights

- **Animation Player** with playback, step controls, keyboard shortcuts, touch gestures, and speed control.
- **Track-based routing** using React Router with lazy-loaded pages.
- **Tailwind CSS + shadcn/ui** components and design tokens for rapid UI assembly.
- **Framer Motion** timeline stubs powering placeholder SVG animations.
- **Zustand store** for player state and timeline updates.
